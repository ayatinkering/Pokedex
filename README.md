# PokéLens

A fun Pokémon discovery tool powered by PokéAPI.

Browse Pokémon through a handcrafted Pokédex interface with smooth pagination, lazy-loaded artwork, detailed stats, and moves information.

## Features

- Search from 1300+ Pokémon instantly
- Detailed Pokémon information modal
- Type badges with unique colors
- Base stats visualized with animated stat bars
- LocalStorage caching for Pokémon data
- Responsive card grid layout
- Smooth page navigation with automatic scroll-to-top
- Pokémon count indicator (e.g. Showing 61–120 of 1350 Pokémon)
- Hover interactions and retro gaming aesthetic

## Built With

- React
- Vite
- Tailwind CSS
- JavaScript
- PokéAPI
